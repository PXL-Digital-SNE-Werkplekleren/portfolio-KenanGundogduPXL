# Overzicht Ontwikkelingsgesprekken

## Overzicht ontwikkelingsgesprekken WPL 1

1. Mijn keuze voor de opleiding

Ik koos voor deze richting omdat ik al 2jaar op het secundair de richting IT volgde en dit vond ik altijd een interessante onderwerp. Ik hou mezelf graag bezig met computers. Dus met deze opleiding te volgen wil ik meer kennis hebben.


2. Mijn beeld van dit beroep

Mijn beeld van dit beroep is dat alles in de toekomst alleen maar IT zal nodig hebben. Meer en meer word er dagelijks gewerkt met computers of dergelijke nieuwe machines of toestellen. Omdat ik het al graag doe wil ik dus met een uitgebreide kennis de IT wereld binnen geraken.

 4. Mijn evolutie tijdens WPL1

Mijn belangrijke leerelementen van WPL1 waren de volgende:

Werkplekleren is een interessante vak en in mijn ogen gaat en heeft dit deels mijn visie bepaald voor het werkleven.
IT organisation heeft me ook veel bijgeleerd over het bedrijfsleven.
Communication skills was voor het 1ste semester simpelder dan verwacht.
Een overzichtelijke planning is iets wat ik voor de komende jaren nog wil meenemen.

Dit zijn mijn sterke punten die ik vorig jaar heb geïdentificeerd:

- IT organisation is een vak waar ik best wel goed in ben. Ik kan door de gegeven leerstof makkelijk dingen afleiden en begrijpen.
- Windows Essentials is een gelijkaardige vak van wat ik op het secundair heb gezien. Voor de basis is dit een voordeel voor mij.

Dit zijn mijn ontwikkelpunten waarin ik tijdens WPL2 beter hoop te worden:

- Linux bijhouden is af en toe nog steeds moeilijk en ik hoop door een andere aanpak het voor de volgend semester het makkelijker te maken voor mijzelf.
- De vragenlijst wat we van Windows Essentials ontvangen wil ik eerder beantwoorden. Soms kan het zijn dat ik het vergeet.

## Groei bij ontwikkelingsgesprekken

In de voorbije periode heb ik duidelijk vooruitgang geboekt, zowel op vlak van kennis als op vlak van soft skills. Door actief te werken aan mijn opdrachten en regelmatig feedback te verwerken, merk ik dat ik sterker ben geworden in mijn manier van werken.

Op kennisvlak heb ik bijgeleerd hoe ik taken gerichter en efficiënter kan aanpakken. Ik begrijp beter wat er van mij verwacht wordt en ik kan zelfstandiger werken dan vroeger. Ook bij nieuwe of moeilijkere opdrachten merk ik dat ik sneller inzicht krijg in wat ik moet doen.

Naast mijn technische kennis ben ik ook gegroeid in mijn soft skills. Ik communiceer vlotter, luister beter naar feedback en probeer daar ook echt iets mee te doen. Daarnaast ben ik kritischer geworden voor mijn eigen werk en denk ik meer na over hoe ik iets beter kan aanpakken.
